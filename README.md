## Ініціалізація гри

* Створюються два ігрових поля.
* Обирається режим гри: **1 vs бота** або **2 гравці**.
* Гравці розміщують кораблі — автоматично або вручну.

## Розміщення кораблів

Для кожного корабля відбувається перевірка можливості розміщення:

* Вибираються випадкові координати та орієнтація (горизонтальна або вертикальна).
* Перевіряється:

  * чи не виходить корабель за межі поля;
  * чи не перетинається з іншими кораблями;
  * чи дотримано мінімальний відступ у 1 клітинку.
* Якщо всі умови виконано — корабель розміщується на полі.

## Ігровий цикл

Гра триває, доки один із гравців не знищить усі кораблі суперника.

### Хід гравця

* Відображаються обидва поля.
* Гравець вводить координати пострілу.
* Перевіряється результат:

  * **Влучання** — гравець продовжує хід.
  * **Промах** — хід переходить до суперника.
* Після кожного пострілу перевіряється, чи є переможець.

### Хід комп’ютера (у режимі vs бота)

* Комп’ютер вибирає ціль:

  * якщо було попереднє влучання — стріляє навколо нього;
  * інакше — обирає випадкову клітинку.
* Після пострілу перевіряється, чи завершена гра.

## Алгоритм бота

* Якщо є попереднє влучання — перевіряються чотири сусідні клітинки та вибирається невідстріляна.
* Якщо влучань немає — робиться випадковий вибір клітинки.
* Після пострілу:

  * у разі влучання координати запам’ятовуються;
  * у разі промаху пам’ять скидається.

## Перевірка перемоги

Проходить перевірка всього поля противника:

* Якщо залишився хоча б один неушкоджений корабель — гра триває.
* Якщо всі кораблі знищені — гравець оголошується переможцем.
